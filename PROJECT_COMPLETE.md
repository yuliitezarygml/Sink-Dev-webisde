## 🎯 ПОРЯДОК ДЕЙСТВИЙ (пошагово)

### ШАГ 1: Установить и запустить
```bash
npm install
npm start
```
Браузер откроется на http://localhost:3000

### ШАГ 2: Открыть файлы
- Откройте `public/data/data.json` в VS Code
- Откройте `public/photos/` в проводнике

### ШАГ 3: Отредактировать текст
В файле `data.json`:
```json
"names": { "first": "Ваше имя", "second": "имя партнера" }
"photographer": "Ваше имя"
"email": "ваша@почта.com"
```

### ШАГ 4: Добавить фото
1. Скопируйте свои фото в `public/photos/`
2. Добавьте пути в `data.json`:
```json
{
  "id": 1,
  "src": "/photos/photo-1.jpg",
  "alt": "Описание",
  "category": "Портреты"
}
```

### ШАГ 5: Проверить
Откройте браузер - сайт должен обновиться автоматически!
