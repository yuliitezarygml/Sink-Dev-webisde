## 🔧 КАК РЕДАКТИРОВАТЬ (ПРИМЕРЫ)

### Изменить имена на главном экране:

Откройте: `src/App.tsx`
```tsx
<Hero 
  names={{ first: 'Ваше имя', second: 'имя партнера' }}
/>
```

### Добавить фото в галерею:

Откройте: `src/components/Gallery/Gallery.tsx`
```tsx
{
  id: 7,
  src: 'https://ваша-ссылка-на-фото.jpg',
  alt: 'Описание',
  category: 'Портреты'
}
```

### Изменить цвета:

Откройте: `src/components/Hero/Hero.css`
```css
background-color: #1a1a2e;  /* ваш цвет */
```

### Изменить контакты:

Откройте: `src/components/Contact/Contact.tsx`
```tsx
<Contact 
  email="ваша@почта.com"
  phone="+7 (999) 123-45-67"
/>
```

```bash
npm install
```

### Сайт не открывается?
- Проверьте, что терминал не показывает ошибки

- Сайт всё равно работать будет
- Если не работает - смотрите консоль браузера (F12)
